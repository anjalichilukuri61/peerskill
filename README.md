# PeerSkill Hub

PeerSkill Hub is a full-stack student skill exchange platform enabling ethical academic support.

## Tech Stack
- **Frontend**: React (Vite)
- **Backend**: Node.js (Express)
- **Database**: Firebase (Firestore, Auth)

## Getting Started

### Prerequisites
- Node.js installed

### Setup

1. **Client**
   ```bash
   cd client
   npm install
   npm run dev
   ```

2. **Server**
   ```bash
   cd server
   npm install
   npm run dev
   ```
